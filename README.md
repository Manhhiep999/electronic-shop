# Electronics Shop Website

## 1. Giới thiệu

Đây là dự án website bán đồ điện tử được xây dựng theo mô hình MVC sử dụng Spring Boot.  
Website cho phép người dùng xem sản phẩm, tìm kiếm, thêm vào giỏ hàng và đặt hàng.  
Admin có thể quản lý sản phẩm, danh mục và đơn hàng.

---

## 2. Công nghệ sử dụng

### Backend

- Java Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL

### Frontend

- HTML
- CSS
- JavaScript
- Bootstrap
- Thymeleaf

### Tools

- Git & GitHub
- IntelliJ IDEA
- MySQL Workbench

---

## 3. Các chức năng chính

### Người dùng

- Đăng ký tài khoản
- Đăng nhập / đăng xuất
- Xem danh sách sản phẩm
- Xem chi tiết sản phẩm
- Tìm kiếm sản phẩm
- Lọc sản phẩm theo danh mục
- Thêm sản phẩm vào giỏ hàng
- Xem giỏ hàng
- Đặt hàng

### Quản trị viên

- Quản lý sản phẩm
- Quản lý danh mục
- Quản lý người dùng
- Quản lý đơn hàng
- Dashboard thống kê

---

## 4. Database

Database sử dụng MySQL với các bảng chính:

- users
- products
- categories
- orders
- order_items
- cart
- cart_items
